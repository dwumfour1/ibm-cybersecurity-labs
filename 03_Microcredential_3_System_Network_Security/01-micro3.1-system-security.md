# Micro 3.1: System Security

## About This Learning Activity
In this module, you'll dive into the fundamentals of **system security**, covering firmware, servers, operating systems, and operating system security practices. 

You will learn to identify vulnerabilities and implement protections that secure systems at multiple levels:

- Explore **firmware vulnerabilities** and understand how securing firmware establishes the foundation for overall system security.  
- Learn about **servers and operating system types**, and how to select and configure them for secure deployments.  
- Discover **operating system security techniques** used to protect systems, applications, databases, and networks in real-world environments.

---

## Learning Objectives
By the end of this module, you should be able to:

1. Update firmware securely  
2. Secure a host operating system  
3. Create a plan to protect operating systems from common threats  

---

## Lessons

- Firmware Security  
- Servers and Operating Systems  
- Operating System Security
