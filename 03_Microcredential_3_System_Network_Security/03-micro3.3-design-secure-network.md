# Micro 3.3: Design a Secure Network for Increased Security

## About This Learning Activity
In this capstone project, you will **design a secure network** to enhance system architectures and increase overall security.  

You will apply your knowledge from **System Security** and **Network Security** to:

- Answer questions about secure network principles and practices  
- Plan a network architecture that strengthens system security  
- Integrate best practices from previous modules into a practical design

---

## Prerequisite Learning
This project builds on the knowledge and skills from:

- Micro 3.1: System Security  
- Micro 3.2: Network Security  

---

## Learning Objectives
By the end of this module, you should be able to:

1. Secure a host operating system  
2. Update firmware on devices  
3. Design a secure network architecture  

---

## Assessment Details
You will complete **two assessments** in this capstone project:

1. **Project Questions:**  
   Demonstrate your knowledge of system and network security. Feedback is provided, and you must score at least 80%. Multiple attempts allowed.

2. **10-Question Assessment:**  
   Test your applied knowledge and skills from the project. A passing score of 80% is required. Feedback is provided, and retries are allowed.

---

## Reflection
This capstone project allowed me to **apply theory to practice**. By designing a secure network and considering system security controls, I reinforced my understanding of layered security, risk mitigation, and network hardening strategies.  

Documenting the process ensures I can **explain and replicate secure network designs** in real-world scenarios.
