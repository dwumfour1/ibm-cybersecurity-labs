# Micro 4.1: Cloud Computing and Virtualization

## About This Learning Activity
In this module, you'll explore the foundations of **cloud computing and virtualization**, focusing on how modern systems are built, deployed, and secured in virtual environments.

You will learn to:

- Understand **system virtualization** and how virtual machines are created, configured, and secured  
- Explore **network virtualization**, and how physical networks are transformed into software-defined environments  
- Learn how **cloud computing** delivers services over the internet  
- Examine **Cloud as a Service (CaaS)** and how organizations use cloud-based infrastructure and applications  

---

## Learning Objectives
By the end of this module, I am able to:

1. Create and configure a virtual machine  
2. Apply the appropriate cloud computing framework based on organizational needs  

---

## Lessons

- System Virtualization  
- Network Virtualization  
- Cloud as a Service  

---

## Reflection
This module introduced me to how modern IT infrastructure relies heavily on **virtualization and cloud technologies**. Understanding how to create and secure virtual machines, as well as how cloud services are structured, is essential for building scalable and secure systems.  

By documenting this, I am strengthening my ability to **work with virtual environments and apply cloud solutions** in real-world cybersecurity scenarios.
