# Micro 5.2: Security Monitoring

## About This Learning Activity
In this module, you'll explore tools and techniques used to **monitor systems and detect security threats** in real time.

You will learn how organizations maintain visibility over their infrastructure:

- Understand **network diagnostic and performance monitoring tools**  
- Explore **endpoint management** and **Endpoint Detection and Response (EDR)**  
- Learn how EDR tools help detect and respond to threats on devices  
- Discover how **Security Information and Event Management (SIEM)** systems improve security  
- Perform a **threat investigation using Splunk Enterprise**  

---

## Learning Objectives
By the end of this module, I am able to:

1. Perform basic network reconnaissance  
2. Apply endpoint management and monitoring principles using EDR tools  
3. Investigate security threats using Splunk Enterprise  

---

## Lessons

- Security Monitoring Tools  
- Endpoint Monitoring  
- Security Information and Event Management (SIEM)

## Reflection
This module introduced me to the importance of **continuous monitoring in cybersecurity**. By using tools like EDR and SIEM, organizations can detect and respond to threats quickly.  

Working with Splunk helped me understand how security data is analyzed in real-world scenarios, strengthening my ability to **investigate and respond to security incidents effectively**.
