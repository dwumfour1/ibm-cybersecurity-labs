# Micro 6.1: Incident Response

## About This Learning Activity
In this module, I explored the fundamentals of **incident response**, focusing on how organizations detect, manage, and recover from security incidents.

I learned how structured response processes help minimize damage and restore normal operations efficiently.

Key areas covered include:

- Understanding the phases of the **incident response lifecycle**  
- Learning the roles and responsibilities within an **incident response team**  
- Developing and applying **incident response plans**  
- Exploring intrusion analysis and attacker behavior frameworks  

---

## Learning Objectives
By the end of this module, I am able to:

1. Apply an appropriate incident response framework based on organizational needs  
2. Identify roles within an incident response team  
3. Apply the MITRE ATT&CK framework  
4. Apply the Cyber Kill Chain framework  
5. Respond effectively to a network attack scenario  

---

## Lessons

- Incident Response Overview  
- Incident Response Planning  
- Intrusion Analysis  

---

## Reflection
This module helped me understand how organizations respond to cyber incidents in a structured and strategic way.  

By studying frameworks like **MITRE ATT&CK** and the **Cyber Kill Chain**, I learned how attackers operate and how defenders can anticipate and disrupt attacks.  

This knowledge strengthens my ability to think like a security analyst and respond effectively under pressure.
