# Part 1: Cybersecurity Final Capstone Project

## About This Learning Activity
This project is the first part of the final IBM SkillsBuild Cybersecurity Certificate capstone project: **Enhance Organizational Cybersecurity and Incident Response**.

In this scenario, I evaluated and strengthened the cybersecurity posture of Bristol and Associates, a law firm affected by a malware attack.

The main focus of this project was to:

- Respond to the cybersecurity incident  
- Contain and mitigate the malware threat  
- Investigate the source of the attack  
- Apply incident response and digital forensics techniques  
- Improve the organization’s overall security posture  

---

## Learning Objectives
By the end of this module, I am able to:

1. Apply an appropriate incident response framework based on organizational needs  
2. Justify threat mitigation tactics  
3. Categorize vulnerabilities by severity to prioritize controls  
4. Secure a host operating system  
5. Update firmware on devices  
6. Investigate incident reports using digital forensics  
7. Create a drive image for forensic analysis  
8. Analyze digital forensic evidence  

---

## Prerequisite Learning
This project builds on my knowledge and skills acquired throughout:

## Assessment Details
You will complete two assessments in this capstone project:

1. **Project Questions:**  
   Demonstrate knowledge of organizational cybersecurity and incident response. A score of at least 80% is required, with multiple attempts allowed.

2. **10-Question Assessment:**  
   Evaluate applied cybersecurity knowledge and skills from the project. A minimum score of 80% is required, with feedback and retry options available.
