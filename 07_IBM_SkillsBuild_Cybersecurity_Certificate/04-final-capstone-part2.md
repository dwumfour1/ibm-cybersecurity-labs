# Part 2: Cybersecurity Final Capstone Project

## About This Learning Activity
This project is the second part of the final IBM SkillsBuild Cybersecurity Certificate capstone project: **Enhance Organizational Cybersecurity and Incident Response**.

In this scenario, I continued working with Bristol and Associates, a law firm affected by a malware attack, focusing on strengthening its cybersecurity posture and preventing future incidents.

The main focus of this project was to:

- Secure the organizational infrastructure  
- Improve network security and system defenses  
- Apply cloud security and deployment strategies  
- Manage endpoints and security tools  
- Configure SOC roles and responsibilities  
- Assess and manage organizational risk  
- Develop backup, recovery, and encryption strategies  

---

## Learning Objectives
By the end of this module, I am able to:

1. Perform network reconnaissance  
2. Design a secure network architecture  
3. Apply appropriate cloud deployment models based on organizational needs  
4. Manage endpoints effectively  
5. Distinguish between security operations models  
6. Assign SOC roles and responsibilities  
7. Assess cybersecurity risk  
8. Analyze the impact of cybersecurity threats on systems and processes  
9. Develop backup and recovery plans  
10. Apply file and drive encryption  

---

## Prerequisite Learning
This project builds on the knowledge and skills acquired throughout:

- Microcredential 1: Governance, Risk, Compliance & Data Privacy  
- Microcredential 2: Vulnerability Management  
- Microcredential 3: System & Network Security  
- Microcredential 4: Cloud Security  
- Microcredential 5: Security Operations & Management

## Assessment Details
You will complete two assessments in this capstone project:

1. **Project Questions:**  
   Demonstrate knowledge of cybersecurity infrastructure and defense strategies. A score of at least 80% is required, with multiple attempts allowed.

2. **10-Question Assessment:**  
   Evaluate applied cybersecurity knowledge and skills from the full program. A minimum score of 80% is required, with feedback and retry options available.
