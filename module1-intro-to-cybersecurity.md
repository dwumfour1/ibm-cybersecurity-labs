# IBM SkillsBuild Cybersecurity Certificate — Module 1: Introduction to Cybersecurity

## Learning Objectives
After completing this module, I am able to:
1. Determine ways in which the three key elements of cybersecurity (Confidentiality, Integrity, Availability — CIA triad) apply to common tasks
2. Categorize the various types of sensitive data that organizations must protect
3. Apply the CIA triad to real-world situations
4. Apply file and drive encryption
5. Develop a plan for backup

## Lessons Covered
- Introduction to cybersecurity
- Data privacy
- Data security
- Data encryption
- Data backup

## Key Takeaways
- Cybersecurity is built around the CIA triad: Confidentiality, Integrity, Availability
- Organizations need to classify and protect sensitive data (PII, financial data, intellectual property)
- Encryption protects files and drives from unauthorized access
- Regular backups are critical to ensure business continuity
- Real-world application requires combining policies, tools, and procedures

## Reflection
This module gave me a solid foundation in understanding core cybersecurity principles and practical measures organizations take to protect data. I can now apply the CIA triad in lab exercises and plan secure storage and backup strategies.
